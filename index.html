HTML Style & Text Formatting

HTML Styles

The HTML style attribute is used to add styles to an element, such as color, font, size, and more.

Property Name Description
background-color Background Color Defines the background color for an HTML element.
color Text Color Defines the text color for an HTML element.
font-family Fonts Defines the font to be used for an HTML.
font-size Text Size Defines the text size for an HTML element.
text-align Text Alignment Defines the horizontal text alignment for an HTML element.

HTML Formatting Elements

HTML contains several elements for defining text with a special meaning.

Element Name Description
<b> Bold Defines Bold text, without any extra importance.
<strong> Strong Defines text with strong importance and displayed in Bold.
<i> Italic Defines a part of text in an alternate voice or mood. The content inside is typically displayed in italic.
<em> Emphasis Defines emphasized text. The content inside is typically displayed in italic.
Note: A screen reader will pronounce the words in <em> with an emphasis, using verbal stress.
<small> Small Defines smaller text.
<mark> Mark Defines text that should be marked or highlighted.
<del> Delete Defines text that has been deleted from a document. Browsers will usually strike a line through deleted text.
<ins> Insert Defines a text that has been inserted into a document. Browser will usually underline inserted text.
<sub> Subscript Defines subscript text. Subscript text appear half a character below the normal line, and is sometimes rendered in a smaller font. Subcript text can be used for chemical formulas, like H2O.
<sup> Superscript Defines superscript text. Superscript text appear half a character above the normal line, and is sometimes rendered in a smaller font. Superscript text can be used for footnotes, like WWW[1].
